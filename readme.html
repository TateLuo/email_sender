Email Sender使用手册

欢迎首次使用Email Sender!

使用须知：

1.在设置页面导入将预存为csv格式的表格（客户数据）导入至本程序，csv表名即为群发邮件的群组名可自定义，文件表头应如下：
company email_info name1 position1 email1 其他联系人依次类推

2.导入数据可后在设置-编辑配置文件中填写配置信息，要求如下：

[stmp_info]
server = 发信(stmp)邮件服务器地址
port = 发信(stmp)邮件服务器端口
username = 你的邮箱地址
password = 你的邮箱密码

[imap_info]
server = 收信（imap）邮件服务器地址
port = 收信（imap）邮件服务器端口
username = 你的邮箱地址
password = 你的邮箱密码

[database]
db_name = 数据库名，默认为customer.db，无法更改
table_name = 你需要发信的数据库名，即为你导入数据时的表名

[schedule]
schedule = 定时发送的时间（可忽略）

[stage_info]
time_gap = 每次发送邮件的间隔，本处填写的间隔数量即为对应的发信次数，同时要有对应数量的模板，模板名数字.html，从0.html开始依次类推。

[user_info]
自定义的变量在邮件编辑器中以{{username}}这种形式引用
customize_variable = {"0":{"subject":"example","username":"example","company":"example","position":"example"}}


3.配置完成后，点击开始便会依次发送，顺序按照表名-表中的顺序发送
